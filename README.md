# Product Management System

Aplikasi CRUD sederhana untuk manajemen produk menggunakan Laravel dan PostgreSQL.

## Tech Stack

- Laravel 10.x  
- PostgreSQL  
- Bootstrap 5  
- PHP 8.1+

## Fitur

- List produk dengan pagination  
- Tambah produk baru  
- Edit produk  
- Hapus produk  
- Validasi form  
- Flash message
